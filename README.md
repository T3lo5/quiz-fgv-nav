# Quiz FGV Navegação Aérea

Um quiz online desenvolvido para estudo e preparação para concursos na área de navegação aérea, baseado no plano de estudos da FGV.

## 🚀 Funcionalidades

- Quiz interativo com questões variadas
- Diferentes níveis de dificuldade (Fácil, Médio, Difícil)
- Sistema de pontuação e feedback imediato
- Modo de revisão com explicações
- Interface responsiva e moderna

## 📚 Disciplinas Cobertas

- **Língua Portuguesa**: Gramática, Ortografia, Interpretação
- **Raciocínio Lógico Matemático**: Lógica, Probabilidade, Equações
- **Informática**: Microsoft Office, Segurança Digital, Redes
- **Legislação**: Aeronáutica Civil, Lei 13.303/2016, Constituição
- **Língua Inglesa**: Gramática, Vocabulário Técnico

## 🎮 Como Usar

1. Abra o arquivo `index.html` no navegador
2. Configure o quiz:
   - Tipo de prova
   - Número de questões
   - Tempo por questão (opcional)
3. Clique em "Iniciar Quiz"
4. Responda as questões clicando nas alternativas
5. Ao final, veja seu resultado e faça a revisão
